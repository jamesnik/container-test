### Instructions:

  - Use any of your preferred language for this test. You have 90 minutes.
  - Put appropriate comments to explain your algorithm and define a set of test cases.


### Description:

  - Given an array of non-negative integers representing height of blocks (black) where they form containers and can hold water (blue) between walls. 
  - For example [0, 1, 0, 2, 1, 0, 1, 3, 2, 0, 1, 0, 2]

### Installation

Requires [Composer](https://getcomposer.org/) to run.

Install the dependencies and devDependencies and start the server.

```sh
$ git clone https://github.com/jamesnik/container-test.git
$ cd container-test
$ composer install
$ php artisan serve
```

Verify the deployment by navigating to your server address in your preferred browser.

```sh
http://localhost:8000 or http://127.0.0.1:8000
```

Or could run immediately on a browser for JavaScript version
```sh
Go to public folder
open container-test-js-version-only.html file on your preferred browser.
```
